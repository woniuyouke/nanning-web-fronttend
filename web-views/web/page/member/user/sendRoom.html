    房源发布须知

    1.

    房源发布者所填写的内容及上传的材料都应真实、合法、有效，如有不实，自行承担由此产生的一切责任和法律后果。

    2.

    房源发布者在填写房源发布信息、房源概况、上传房源图片时，请认真核对，房源发布成功后，房源信息将直接在平台展示；如需修改房源信息，可将房源下架后重新编辑。

    3.

    平台会对所有发布的房源进行系统核验，核验通过的房源会生成唯一房源核验码，用手机扫描该房源核验码即可查询到所发布的对应房源详情。

    4.

    已通过核验的房源，在房源详情页会展示其“核验”、“抵押”、“限制”、“在租”的状态。
